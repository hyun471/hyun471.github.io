## ChatGPT Assignment

- Shoppingmall 처럼 Console로 작동하는 조금 더 어려운 문제를 내달라고 하였다.
### Console Book Rental System

- 목표 : 도서 대여 시스템 만들고 아래 기능 구현하기
**기능 요구 사항  : 다음 사항을 사용하여 만들기**

[1]  도서 목록 보기

- book 클래스의 객체들이 List<Book>에 저장되어 있음
- 각 책은 제목, 저자, 가격을 가짐
- 전체 목록을 보여줌
[2] 도서 대여하기

- 제목을 입력받음
- 존재하는 도서만 대여 가능
- 대여 중인 책이면 “이미 대여 중입니다.” 출력
- 대여 가능하면 “대여 완료!” 출력
[3] 대여 현황 보기

- 대여된 도서 목록을 모두 출력
- 대여된 책이 없으면 “대여된 책이 없습니다.” 출력
[4] 프로그램 종료

- 프로그램 종료
**조건 : **

- 책 제목 중복은 없음
- 대여된 책은 상태가 유지되어야 함
- 예시출력
```dart
[1] 도서 목록 보기 / [2] 도서 대여하기 / [3] 대여 현황 보기 / [4] 종료
입력값: 2
```

## 트러블슈팅 과정

### Book() 생성자 호출 시 에러

- 문제 : Book volume = Book(); 코드를 작성했더니 괄호 쪽에서 에러가 발생
- 원인 : Book 클래스에 기본 생성자가 없고, 파라미터를 요구하는 생성자만 있어서 에러가 발생
- 해결 : Book(”제목”, “저자”, 가격) 처럼 생성자 인자를 넣어줌
### Map<Book, int>사용 시 의도대로 비교되지 않음

- 문제 : Book 객체를 key로 쓰는 Map에서 중복 비교나 포함 여부가 제대로 작동하지 않음
- 원인 : Map를 사용하게 되면 value값도 따로 지정을 해줘야 하고 ==와 hashCode를 재정의 해줘야함
- 해결 : 같은 Collection이고 중복 값도 제거가 가능하고 대여 여부만 필요하기 때문에 Set<Book> 사용
### 추가적으로 넣고 싶은 기능

- 도서 반납 기능을 추가해보고 싶다.  
**코드 설계**

  - 도서를 대여하면 myRental과 totalRental에 들어가게 된다.
  - 반납 메서드를 실행하면 반납할 도서명을 입력한다.
  - 입력된 도서명과 myRental에 있는 도서명이 일치하면 myRental에 있는 도서명을 지운다.
  - 일치하지 않으면 “도서명이 올바르지 않습니다.” 출력
