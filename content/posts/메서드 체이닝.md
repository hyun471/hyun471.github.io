# 메서드 체이닝

# 📚 함수형 프로그래밍 1

## 메서드 체이닝(함수의 연속)

- 하나의 객체에 여러 메서드를 점(.)을 이용해 연속적으로 호출하는 프로그래밍 방식이다.
- 즉, 메서드 결과를 다시 객체로 반환해서 계속 연결하는 구조이다.
- 메서드 체이닝은 함수를 정의하는게 아니라 연속으로 호출하는 구조이다.
- 메서드 체이닝은 객체와 메서드 반환값이 있어야하고 점연산자로 연결하여 호출한다.
  - 여기서 객체(number)가 abs()의 반환값을 받고 다시 toString() 반환값을 받고 다시 contains(’3’)의 반환값을 받아서 result에 넣음
```dart
int number = -12345                                   // 변수 선언
var result = number.abs().toString().contains('3');   // 연속으로 함수 호출
print(result);                                        // 출력 : true

// abs(), toString(), contains() 모두 Dart에서 기본 제공하는 내장 함수
// abs()는 절댓값을 반환하는 함수,
// toString()은 문자형으로 바꿔주는 함수,
// contains()는 문자열에 특정 문자가 있는지 검사하는 함수
```

- 임의의 함수로 메서드 체이닝을 하려면 그 함수가 객체의 메서드로 정의되어야 하며, 반환값도 체이닝이 가능하도록 객체 자신(this)이나 연결 가능한 값이 있어야 한다.
```dart
class Person {
  String name = '';
  int age = 0;

  // 메서드 1
  Person setName(String name) {
    this.name = name;
    return this; // 객체 자신 반환 (체이닝 가능!)
  }

  // 메서드 2
  Person setAge(int age) {
    this.age = age;
    return this; // 객체 자신 반환
  }

  // 메서드 3
  void introduce() {
    print('이름: $name, 나이: $age');
  }
}

void main() {
  Person()
    .setName('초코')
    .setAge(3)
    .introduce();          // 출력 => 이름 : 초코, 나이 : 3
}
```

### 순수 함수

- 같은 입력 값에 대해 항상 같은 출력 값이 나오고 외부 상태 변경(부작용)이 없는 함수이다.
- 예측이 가능하고 테스트가 쉬우며 안정적이다.
```dart
int add(int a, int b) {             // 순수 함수
  return a + b;                     // 동일한 입력값에 항상 동일한 결과값
}
```

### 형변환 함수

- 데이터의 타입을 다른 타입으로 바꾸는 함수
- 자주 사용되는 형변환 함수
  - 예시 코드
```dart
void main() {
  int number = 123;
  String str = number.toString();     // int → String
  print(str);                         // 출력: "123"

  double d = 3.14;
  int i = d.toInt();                  // double → int
  print(i);                           // 출력: 3

  String s = "42";
  int x = int.parse(s);              // String → int
  print(x + 1);                       // 출력: 43

  List<String> fruits = ["apple", "banana"];
  String result = fruits.join(", "); // List → String
  print(result);                     // 출력: "apple, banana"
}
```

### 고차 함수

- 함수를 인자로 받거나, 함수를 반환하는 함수. 즉, 함수를 값처럼 다루는 함수이다.
- 자주 사용되는 고차 함수 종류
- map() : 각 요소를 특정 함수에 적용하여 새 Collection 타입의 데이터로 반환한다.
  - 구성 요소 : map(([매개변수]) { return [매개변수에 적용할 동작] });
  - 동작이 한 줄로 표현 가능한 경우 : map(([매개변수]) => [매개변수에 적용할 동작] );
```dart
List<int> numbers = [1, 2, 3];
var doubledNumbers = numbers.map((n) {
  var x = n * 2;
  var plus = n + 2;
  x += plus;
  return x;
});
print(doubledNumbers);     // 출력 : (5, 8, 11)

List<int> numbers = [1, 2, 3];
var doubledNumbers = numbers.map((n) => n * 2);
print(doubledNumbers);     // 출력 : (2, 4, 6)
```

- where() : 각 요소를 특정 조건에 대해 참인 값을 새 Collection 타입의 데이터로 반환한다.
  - 구성 요소 : where(([매개변수]) { return [조건식] });
  - 조건이 한 줄로 표현 가능한 경우 : where(([매개변수]) => [조건식] );
```dart
List<int> numbers = [1, 2, 3];
var result = numbers.where((n) {
  return n > 2;
});
print(result);     // 출력 : (3)

List<int> numbers = [1, 2, 3];
var result = numbers.where((n) => n > 2);
print(result);     // 출력 : (3)
```

- firstWhere() : 각 요소를 특정 조건에 대해 참인 값 중 첫 번째 요소를 반환한다.
  - 구성 요소 : firstWhere(([매개변수]) { return [조건식] });
  - 조건이 한 줄로 표현 가능한 경우 : firstWhere(([매개변수]) => [조건식] );
```dart
List<int> numbers = [1, 2, 3];
var result = numbers.firstWhere((n) {
  return n < 3;
});
print(result);     // 출력 : (1)

List<int> numbers = [1, 2, 3];
var result = numbers.firstwhere((n) => n < 3);
print(result);     // 출력 : (1)
```

- lastWhere() : 각 요소를 특정 조건에 대해 참인 값 중 첫 번째 요소를 반환한다.
  - 구성 요소 : lastWhere(([매개변수]) { return [조건식] });
  - 조건이 한 줄로 표현 가능한 경우 : lastWhere(([매개변수]) => [조건식] );
```dart
List<int> numbers = [1, 2, 3];
var result = numbers.lastWhere((n) {
  return n < 3;
});
print(result);     // 출력 : (2)

List<int> numbers = [1, 2, 3];
var result = numbers.lastWhere((n) => n < 3);
print(result);     // 출력 : (2)
```

- reduce() : Collection 타입의 모든 요소를 하나의 값으로 결합한다.
  - 구성 요소 : reduce(([매개변수1], [매개변수2]) { return [적용할 동작] });
  - 동작이 한 줄로 표현 가능한 경우 : reduce(([매개변수1], [매개변수2]) => [적용할 동작] );
```dart
List<int> numbers = [1, 2, 3];
var result = numbers.reduce((a, b) {
	return a + b;
});
print(result);     // 출력 : 6

List<int> numbers = [1, 2, 3];
var result = numbers.reduce((a, b) => a + b);
print(result);     // 출력 : 6
```

  - 처음 실행엔 초기값을 데이터의 첫 번째 요소를 사용하고 이후에 매개변수1에는 이전 실행에서 반환된 값이 할당되고, 매개변수2에는 Collection 타입의 데이터에 있는 다음 값이 할당됨
  - 같은 타입의 데이터만 반환할 수 있음
- fold() : Collection 타입의 모든 요소를 하나의 값으로 결합한다.
  - 구성 요소 : fold(초기값, ([매개변수1], [매개변수2]) { return [적용할 동작] });
  - 한 줄로 표현 가능한 경우 : fold(초기값, ([매개변수1], [매개변수2]) => [적용할 동작] );
```dart
List<int> numbers = [1, 2, 3];
var result = numbers.fold(0, (a, b) {
	return a + b;
});
print(result);     // 출력 : 6

List<int> numbers = [1, 2, 3];
var result = numbers.fold(0, (a, b) => a + b);
print(result);     // 출력 : 6
```

  - 초기값을 정해주고 이후에 매개변수1에는 이전 실행에서 반환된 값이 할당되고, 매개변수2에는 Collection 타입의 데이터에 있는 다음 값이 할당됨
  - Collection 타입의 데이터와 다른 타입으로도 반환이 가능함
- any() : 요소 중 하나라도 조건에 만족하면 true를 반환한다.
  - 구성 요소 : any(([매개변수]) { return [조건식] });
  - 조건이 한 줄로 표현 가능한 경우 : any(([매개변수]) => [조건식] );
```dart
List<int> numbers = [1, 2, 3];
var result = numbers.any((n) {
  return n < 3;
});
print(result);     // 출력 : true

List<int> numbers = [1, 2, 3];
var result = numbers.any((n) => n < 3);
print(result);     // 출력 : true
```

- every() : 모든 요소가 조건에 만족하면 true를 반환한다.
  - 구성 요소 : every(([매개변수]) { return [조건식] });
  - 조건이 한 줄로 표현 가능한 경우 : every(([매개변수]) => [조건식] );
```dart
List<int> numbers = [1, 2, 3];
var result = numbers.every((n) {
  return n < 3;
});
print(result);     // 출력 : false

List<int> numbers = [1, 2, 3];
var result = numbers.every((n) => n < 3);
print(result);     // 출력 : false
```

- takeWhile() : 조건이 참이 되는 동안 요소를 반환하고 거짓이면 멈춘다.
  - 구성 요소 : takeWhile(([매개변수]) { return [조건식] });
  - 조건이 한 줄로 표현 가능한 경우 : takeWhile(([매개변수]) => [조건식] );
```dart
List<int> numbers = [1, 2, 3, 0];
var result = numbers.takeWhile((n) {
  return n < 3;
});
print(result);     // 출력 : (1, 2)

List<int> numbers = [1, 2, 3, 0];
var result = numbers.takeWhile((n) => n < 3);
print(result);     // 출력 : (1, 2)
```

- skipWhile() : 조건이 참인 동안은 건너뛰고, 거짓이 되는 순간부터 요소를 반환한다.
  - 구성 요소 : skipWhile(([매개변수]) { return [조건식] });
  - 조건이 한 줄로 표현 가능한 경우 : skipWhile(([매개변수]) => [조건식] );
```dart
List<int> numbers = [1, 2, 3, 0];
var result = numbers.skipWhile((n) {
  return n < 3;
});
print(result);     // 출력 : (3, 0)

List<int> numbers = [1, 2, 3, 0];
var result = numbers.skipWhile((n) => n < 3);
print(result);     // 출력 : (3, 0)
```
