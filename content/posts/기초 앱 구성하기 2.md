**실린더 위젯**

- 음량 조절 등과 같이 슬라이더로 조절이 필요할 때 사용
```dart
import 'package:flutter/material.dart';

class SliderBox extends StatefulWidget {
  @override
  State<SliderBox> createState() => _SliderBoxState();
}

class _SliderBoxState extends State<SliderBox> {
  double value = 173;
  @override
  Widget build(BuildContext context) {
    return Column(
      children: [
        Slider(
          value: value, 
          onChanged: (v) {
          // 슬라이더가 움직일 때 이 함수가 실행됨
          // v는 변경된 현재 슬라이더가 있는 값
            setState(() {
              value = v;
            });
          },
          min: 1,
          max: 300,
        )
      ],
    );
  }
}

```

**CircularProgressIndicator을 사용하여 ResultGauge 구현**

- CircularProgressIndicator : 주로 로딩 애니메이션을 나타낼 때 많이 사용
- 0~1까지의 value 속성을 이용하여 해당 값에서 멈춘 위젯으로 사용이 가능
```dart
SizedBox.square(
          dimension: 250,
          child: CircularProgressIndicator(         
            value: 1,                                // 1까지 꽉찬 로딩
            strokeWidth: 2,                          // 두께
            color: Theme.of(context).dividerColor,   // 색상
          ),
        ),
```
