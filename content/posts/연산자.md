# 📚 연산자

## 연산자란?

- 프로그램 내에서 계산, 논리적 비교 등 데이터를 조작하거나 사용하기 위해 사용되는 기호 또는 키워드
## 연산자의 종류

### 산술 연산자

- 덧셈, 뺄셈 등 기본 연산을 수행할 때 사용
- 예시 코드
```dart
void main() {
  int five = 5;
  int two = 2;
  print(five + two);    // 7     더하기
  print(five - two);    // 3     빼기
  print(five * two);    // 10    곱하기
  print(five / two);    // 2.5   나누기
  print(five ~/ two);   // 2     나누고 몫 값
  print(five % two);    // 1     나누고 나머지 값
}
```

### 비교 연산자

- 2개의 값을 비교할 때 사용 
- 예시 코드
```dart
void main() {
  int a = 5;
  int b = 2;
  int c = 2;
  print(a == b);        // false    (5 = 2 같지 않음)
  print(a != b);        // true     (5 ≠ 2 같지 않음)
  print(a > c);         // true     (5 > 2 왼쪽이 큼)
  print(a < b);         // false    (5 < 2 왼쪽이 큼)
  print(a >= b);        // true     (5 ≥ 2 왼쪽이 큼)
  print(b <= c);        // true     (2 ≤ 2 둘이 같음)
}
```

- 타입 체크 연산자
  - 프로그램 실행 중 런타임에 타입을 검사할 때 사용함
  - 예시 코드
```dart
void main() {
  int a = 5;
  int? b = null;
  String c = '2';
  print(a as double);     // 5         (정수 숫자 -> 실수 숫자 타입 변환)
  print(c as double);     // TypeError (문자 타입을 숫자 타입으로 변형 불가)
  print(a is int);        // true      (int 타입 값 존재)
  print(b is! String);    // true      (String 타입 값 존재 하지 않음)
}
```

### 대입 연산자

- 값을 변수에 대입할 때 사용
- 예시 코드
```dart
void main() {
  int a = 5;
  int? b = null;
  int c = 2;
  num d = 3;
  print(a);              // 5    (a = 5)
  print(b ??= c);        // 2    (b = null, c = 2, 2값을 b에 대입)
  b += c;                
  print(b);              // 4    (b = 2, c = 2, b = b + c)
  b -= c;                
  print(b);              // 2    (b = 4, c = 2, b = b - c)
  print(c *= b);         // 4    (c = 2, b = 2, c = c * b)
  print(d /= a);         // 0.6  (d = 3, a = 5, d = d / a)
  print(a ~/= c);        // 1    (a = 5, c = 4, a = a ~/= c)
  print(a %= c);         // 1    (a = 1, c = 4, a = a %= c)
}
```

### 논리 연산자

- 참 또는 거짓을 판별할 수 있는 표현식을 반전하거나 결합할 때 사용
- 예시 코드
```dart
void main() {
  bool a = true;
  bool b = true;
  bool c = false;
  print(!a);          // false    (a = true => 반전)
  print(a || c);      // true     (a = true, c = false, => true)
  print(a && b);      // true     (a = true, b = true, => false)
}
```
